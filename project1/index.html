WhatCanICook

Application definition statement

A recipe finder that can suggest available recipes according to the ingredients people have.

Web services

Food API: https://spoonacular.com/food-api

Open Food Facts API: https://world.openfoodfacts.org/data

Full list of features

1. Manage the list of current ingredients (add, delete, save).
2. List the available recipes according the list of current ingredients.
3. Sort the results by the availability of current ingredients.
4. Search recipes users want and see which ingredients are owned and which are required.
5. Default page will show top popular recipes.

Filtered list of features

1. Manage the ingredients
2. Suggest the recipe according to ingredients
3. Search the recipe
